### 1. Install dbdocs via terminal

First, ensure that `npm` is installed on your Linux system:

```sh
sudo apt update
sudo apt install npm -y

sudo npm install -g dbdocs

dbdocs --version
```

### 2. Define your database with DBML
``` sh
database.dbml
```

### 3. Login to dbdocs
